# Changelog
All notable changes to this project are documented in this file.

## [1.0.0] - 2026-02-17
### Added
- Stable demo baseline for backend, frontend, auth, campaigns, admin, and reporting features.
- Docker-based local environment for reproducible demo startup.
- CI workflow and initial test suite scaffolding.

### Fixed
- Frontend form input remount issue that caused one-character typing behavior.
- React frontend proxy configuration for Docker networking.
- Docker Compose service definition issue for frontend placement.

### Notes
- This version is intended as the first client-demo stable release.
- Use feature branches for all future work and tag each client-ready milestone.
