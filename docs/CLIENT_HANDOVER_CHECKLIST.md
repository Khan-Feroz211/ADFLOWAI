# Client Handover Checklist

## Access and Credentials
- [ ] Production/staging URLs shared
- [ ] Admin account verified
- [ ] `.env` values handed over securely

## Deployment and Operations
- [ ] Deployment method documented (Docker/hosted)
- [ ] Startup/shutdown commands verified
- [ ] Backup and rollback method documented

## Product Validation
- [ ] Login flow verified
- [ ] Campaign creation/edit/delete verified
- [ ] Admin panel verified
- [ ] Reports export verified

## Support and Governance
- [ ] SLA/support plan shared (`docs/SLA_SUPPORT.md`)
- [ ] Ticket workflow agreed
- [ ] Version/tag for handover recorded
